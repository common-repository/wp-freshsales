== Description ==

Elevate your WordPress website to a new level of lead capture and CRM integration with the Freshsales Integration for WordPress plugin. Say goodbye to complex HTML forms and visible public IDs. With WP Freshsales, you can create, manage, and update web forms using dynamic shortcodes, enabling you to embed them anywhere in your WordPress content, from posts and pages to custom posts, sidebars, widgets, and popups. Ensure security with built-in Captcha protection to combat spam effectively.

**Note:** WP Freshsales is now a part of the [WP Leads Builder for any CRM](https://www.example.com/wp-leads-builder-any-crm-pro.html) plugin, encompassing Freshsales CRM, Salesforce, Vtiger, Zoho, and SugarCRM integration features.

With WP Freshsales, you can:

* Generate web forms without the need for HTML or visible public IDs.
* Create dynamic shortcodes for use anywhere in your WordPress content.
* Streamline form updates with a single shortcode edit.
* Easily set up fields by pulling them from Freshsales CRM modules.
* Customize form elements effortlessly through a user-friendly interface.
* Assign forms to available Freshsales users for efficient lead management.

In essence, WP Freshsales empowers you to generate web forms via a user-friendly CRM WordPress plugin interface, turning your website's visitors into high-potential leads for Freshsales CRM.

**Key Features**

* Generate Freshsales WordPress forms with or without web forms.
* Seamlessly integrate with the Contact Form 7 plugin, offering a familiar user experience.
* Utilize Freshsales WordPress plugin forms in pages, posts, popups, opt-ins, and widgets.
* Capture potential leads and sync them with Freshsales CRM.
* Easily redirect users to the desired page after form submission.
* Receive notifications on lead capture success or failure.
* Convert new users into CRM leads with Freshsales plugins.

Elevate your lead capture game and streamline your CRM integration with Freshsales Integration for WordPress. Transform your website into a lead-generating powerhouse effortlessly.

== Installation ==

1. First, install [CRM WordPress Leads](https://wordpress.org/plugins/wp-leads-builder-any-crm)
2. Then Extract the wp-freshsales.zip in wordpress/wp-content/plugins using FTP or through plugin install in wp-admin.
3. Activate the plugin in the WordPress plugin list.
